Existen 3 Tipos de Menús.
Todo depende del usuario con el que se ingrese. Si usted ingresa con un usuario del propietario le aparecerá este menú; es el menú que tiene acceso a todo el sistema.

Botón de Menú Administrativo

Este botón nos llevara al menú administrativo.

Botón Cerrar Sesión
