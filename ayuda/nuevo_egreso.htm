Motivo del Egreso

En este combo se puede seleccionar el motivo del egreso ya sea un pago de salario u otra necesidad de la institución.
si selecciona pago de salario luego debe presionar el botón buscar empleado para que los datos se carguen automáticamente.

Número de identidad del Cliente

En este campo se debe de ingresar el número de identidad de la persona a quien se le esta dando el egreso.

Nombre completo del cliente

En este campo se debe de ingresar el nombre completo de la persona a quien se le esta dando el egreso.

Monto total

Arriba

Buscar Empleado

En caso que se quiera pagar un sueldo entonces se presiona este botón para poder buscar el empleado a quien se desea pagar su sueldo.
