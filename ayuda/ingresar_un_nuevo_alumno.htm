Ingresar Un Nuevo Alumno

En este formulario se nos piden todos los datos del alumno y del padre que se desea registrar en la institución.

Número de Identidad

En este espacio ingresamos el número de identidad del alumno sin guiones. este campo es obligatorio.
Arriba

Primer Nombre del Alumno

En este espacio ingresamos el primer nombre del alumno. Este campo es obligatorio.
Arriba

Segundo Nombre del Alumno

En este espacio ingresamos el Segundo nombre del alumno. Este campo no es obligatorio porque no todas las personas tienen un segundo nombre, por tanto se puede dejar vacío.
Arriba

Primer Apellido del Alumno

En este espacio ingresamos el primer apellido del alumno. Este campo es obligatorio.
Arriba

Segundo Apellido del Alumno

En este espacio ingresamos el segundo apellido del alumno. Este campo no es obligatorio porque no todas las personas tienen un segundo apellido, por tanto se pude dejar vacío.
Arriba

Genero del Alumno

En este combo seleccionamos el genero del alumno que se esta registrando. Este campo es obligatorio.
Arriba

Fecha de Nacimiento del Alumno

En este calendario se selecciona la fecha de nacimiento del alumno que se esta registrando. este campo es obligatorio.
Arriba

Edad del Alumno

Este campo se llena automáticamente al seleccionar la fecha de nacimiento del alumno que se esta ingresando.
Arriba

Estado del Alumno

Este combo nos permite seleccionar el estado del alumno sea activo si sigue en la institución e inactivo si ya no sigue en la institución. si el alumno se ingresa por primera vez no nos permitirá modificarlo ya que como es primer ingreso el alumno esta activo.
Arriba

Repite el grado el alumno

en este control nos permite decidir si el alumno esta repitiendo el grado o no.
Arriba

Grado a Cursar del Alumno

En este combo nos permite seleccionar el grado que cursa el alumno actualmente.
Arriba

Valor de Mensualidad

En este campo no nos permite ingresar nada ya que cuando se selecciona el grado en el que se matricula el alumno, automáticamente se establece el precio de dicho grado ya establecido por la institución.
Arriba

Estado de Salud del Alumno

En este campo debemos ingresar todos los datos relacionados con la salud del estudiante, como si padece de alguna enfermedad o es alérgico a algo, para así estar seguros en caso de una emergencia.
Arriba

Numero de Indentidad del Padre o Encargado

En este espacio ingresamos el número de identidad del padre o encargado sin guiones. este campo es obligatorio.
Arriba

Primer Nombre Padre o Encargado

En este espacio ingresamos el primer nombre del padre o encargado del alumno. Este campo es obligatorio.
Arriba

Segundo Nombre del Padre o Encargado

En este espacio ingresamos el Segundo nombre del padre o encargado del alumno. Este campo no es obligatorio porque no todas las personas tienen un segundo nombre, por tanto se puede dejar vacío.
Arriba

Primer Apellido del Padre o Encargado

En este espacio ingresamos el primer apellido del padre o encargado. Este campo es obligatorio.
Arriba

Segundo Apellido del Padre o Encargado

En este espacio ingresamos el segundo apellido del padre o encargado. Este campo no es obligatorio porque no todas las personas tienen un segundo apellido, por tanto se pude dejar vacío.
Arriba

Genero del Padre o Encargado

En este combo seleccionamos el genero del padre o encargado que se esta registrando. Este campo es obligatorio.
Arriba

Fecha de Nacimiento del Padre o Encargado

En este calendario se selecciona la fecha de nacimiento del padre o encargado que se esta registrando. este campo es obligatorio.
Arriba

Edad del Padre o Encargado

Este campo se llena automáticamente al seleccionar la fecha de nacimiento del padre o encargado que se esta ingresando.
Arriba

Número Móvil del Padre o Encargado

En este campo se debe ingresar el número móvil con el cual se pueda contactar la institución en caso de una emergencia ú otra actividad que lo amerite. este campo no es obligatorio ya que no todas las personas poseen un celular.
Arriba

Correo Electrónico del Padre o Encargado

En este campo colocamos el correo electrónico del padre o encargado con el cual se pueda contactar la institución en caso de una emergencia ú otra actividad que lo amerite. este campo no es obligatorio ya que no todas las personas poseen un correo.
Arriba

Dirección del Padre o Encargado

En este campo colocamos la dirección del padre o encargado, por si sucede alguna emergencia o alguna situación que lo amerite se busca en esa dirección. Este campo es obligatorio.
Arriba

Botón Regresar

Este botón nos lleva al formulario que posee el menú administrativo.
Arriba

Botón Guardar Datos

Una vez ingresado todos los datos podemos proceder a presionar el botón guardar para registrar el alumno en la base de datos de la institución.
Arriba

Botón Actualizar

El boton actualizar nos permite, modificar un dato de un alumno. En este caso nos aparece inhabilitado porque no se ha traido ningún dato de la tabla de reportes de alumnos.
