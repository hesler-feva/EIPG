Reporte Alumnos

En este formulario podremos observar todos los datos relacionados con el alumno.

Búsqueda de Alumno

En este campo podemos buscar el alumno ya sea por nombre o por número de identidad; si se busca por numero de identidad es necesario incluir los guiones. este campo es necesario para realizar una búsqueda.
Arriba

Botón Buscar

Una vez ingresados los datos del alumno que deseamos buscar, presionamos el botón buscar para que nos muestre todos los alumnos que se relacionan con los datos ingresados.
Arriba

Botón Refrescar

Una vez realizada un búsqueda solo nos mostrara los datos relacionados, este botón nos permite refrescar el formulario para poder observar todos los alumnos otra vez.
Arriba

Alumnos Con saldo Pendientes

Si se Selecciona alumnos con saldo pendiente, nos mostrara una lista solo de los alumnos que poseen un saldo pendiente con la institución.
Arriba

Datos de los Alumnos

En este espacio podemos observar todos los datos de los alumnos, con la barra de abajo nos podemos desplazar hacia la derecha para ver mas datos relacionados con el alumno. si se desea modificar un dato solo damos doble clic sobre el alumno que deseamos modificar.
Arriba

Botón Regresar

Este botón nos lleva al formulario que posee el menú administrativo.
Arriba

Botón Generar Reporte

Este botón nos permite generar un reporte que se puede imprimir sobre todos los alumnos de la institución o sobre un alumno en especifico.
Arriba

Nombre del Empleado

En esta parte nos muestra el nombre del empleado que esta activo en ese momento.
Arriba

Cargo del Empleado

En esta parte nos muestra el nombre del cargo que ocupa el empleado dentro del la empresa.
Arriba
