Ingresar Un Nuevo Empleado

En este formulario se nos piden todos los datos del empledao que se desea registrar en la institución.

Número de Identidad del Empleado a registrar

En este espacio ingresamos el número de identidad del empleado sin guiones. este campo es obligatorio.
Arriba

Primer Nombre del Empleado

En este espacio ingresamos el primer nombre del empleado. Este campo es obligatorio.
Arriba

Segundo Nombre Empleado

En este espacio ingresamos el Segundo nombre del empleado. Este campo no es obligatorio porque no todas las personas tienen un segundo nombre, por tanto se puede dejar vacío.
Arriba

Primer Apellido del Empleado

En este espacio ingresamos el primer apellido del empleado. Este campo es obligatorio.
Arriba

Segundo Apellido del Empleado

En este espacio ingresamos el segundo apellido del empleado. Este campo no es obligatorio porque no todas las personas tienen un segundo apellido, por tanto se pude dejar vacío.
Arriba

Genero del Empleado

En este combo seleccionamos el genero del empleado que se esta registrando. Este campo es obligatorio.
Arriba

Fecha de Nacimiento del Empleado

En este calendario se selecciona la fecha de nacimiento del empleado que se esta registrando. este campo es obligatorio.
Arriba

Edad del Empleado

Este campo se llena automáticamente al seleccionar la fecha de nacimiento del empleado que se esta ingresando.
Arriba

Número Móvil del Empleado

En este campo se debe ingresar el número móvil con el cual se pueda contactar la institución en caso de una emergencia ú otra actividad que lo amerite. este campo no es obligatorio ya que no todas las personas poseen un celular.
Arriba

Estado Civil del Empleado

en este combo seleccionamos el estado civil del empleado. Es un campo obligatorio.
Arriba

Correo Electrónico del Empleado

En este campo colocamos el correo electrónico del empleado con el cual se pueda contactar la institución en caso de una emergencia ú otra actividad que lo amerite. este campo no es obligatorio ya que no todas las personas poseen un correo.
Arriba

Dirección del Empleado

En este campo colocamos la dirección del empleado, por si sucede alguna emergencia o alguna situación que lo amerite se busca en esa dirección. Este campo es obligatorio.
Arriba

Titulo del Empleado

En este campo colocamos el titulo que posee el empleado actualmente. campo obligatorio.
Arriba

Post-grado del empleado

este campo es una decisión si el empleado posee un post grado.
Arriba

Post grado del Empleado

si el empleado posee un post grado lo colocamos, si no se deja vacío ya que no es un campo obligatorio.
Arriba

Experiencia Laboral

En este campo colocamos la experiencia laboral que tiene el empleado. Este campo puede quedar vacío ya que no todas las personas tienen una experiencia laboral.
Arriba

Estado del Empleado

Esta combo nos permite establecer el estado del empleado ya sea activo o inactivo. En este caso aparace inhabilitado porque se esta registrando un empleado y por tanto es activo.
Arriba

Salario del Empleado

En este campo colocamo el monto total que se le pagara al empleado. Este campo es obligatorio.
Arriba

Nivel de Acceso del Empleado

Este campo es muy importate ya que aqui se coloca el cargo que ocupara el empleado; si es un empleado administrativo o contable tendra un usuario para acceder al sistema. Este campo es obligatorio.
Arriba

Usuario del Empleado

Si el nivel de acceso del empleado es Administrativo o Contable entonces se coloca el nombre del usuario con el que tendrá acceso al sistema.
Arriba

Contraseña del Empleado

Si el nivel de acceso del empleado es Administrativo o Contable entonces se coloca la contraseña del usuario con la que tendrá acceso al sistema
Arriba

Botón Regresar

Este botón nos lleva al formulario que posee el menú administrativo.
Arriba

Botón Guardar

Una vez ingresado todos los datos podemos proceder a presionar el botón guardar para registrar el empleado en la base de datos de la institución.
Arriba

Botón Actualizar

El boton actualizar nos permite, modificar un dato de un empleado. En este caso nos aparece inhabilitado porque no se ha traido ningún dato de la tabla de reportes de empleados.
Arriba
