Botón Buscar Alumno

Este botón nos lleva al reporte de alumnos para asi poder seleccionar el alumno a quien le queremos pagar su matricula o mensualidad.

Número de Identidad

Una vez seleccionado el alumno que se desea pagar su matricula o mensualidad, este campo se llenará de manera automática con el número de identidad del alumno seleccionado.

Nombre Completo del Alumno

Una vez seleccionado el alumno que se desea pagar su matricula o mensualidad, este campo se llenará de manera automática con el nombre completo del alumno seleccionado.

Grado Cursando del Alumno

Una vez seleccionado el alumno que se desea pagar su matricula o mensualidad, este campo se llenará de manera automática con el grado cursando del alumno seleccionado.

Número de identidad del padre

Una vez seleccionado el alumno que se desea pagar su matricula o mensualidad, este campo se llenará de manera automática con el número de identidad del padre de dicho alumno seleccionado.

Nombre Completo del Padre

Una vez seleccionado el alumno que se desea pagar su matricula o mensualidad, este campo se llenará de manera automática con el nombre completo del padre de dicho alumno seleccionado.

Dirección del Padre

Una vez seleccionado el alumno que se desea pagar su matricula o mensualidad, este campo se llenará de manera automática con la dirección del padre de dicho alumno seleccionado.

Una vez seleccionado el alumno que se desea pagar su matricula o mensualidad, este campo se llenará de manera automática con el saldo pendiente.
