Reportes Empleados

En este formulario podremos observar todos los datos relacionados con el empleado.

Búsqueda de Empleado

En este campo podemos buscar el empleado ya sea por nombre o por número de identidad; si se busca por numero de identidad es necesario incluir los guiones. este campo es necesario para realizar una búsqueda.
Arriba

Botón Buscar

Una vez ingresados los datos del empleado que deseamos buscar, presionamos el botón buscar para que nos muestre todos los empleados que se relacionan con los datos ingresados.
Arriba

Botón Refrescar

Una vez realizada un búsqueda solo nos mostrara los datos relacionados, este botón nos permite refrescar el formulario para poder observar todos los empleados otra vez.
Arriba

Datos del Empleado

En este espacio podemos observar todos los datos de los empleados, con la barra de abajo nos podemos desplazar hacia la derecha para ver mas datos relacionados con el empleado. si se desea modificar un dato solo damos doble clic sobre el empleado que deseamos modificar.
Arriba

Botón Regresar

Este botón nos lleva al formulario que posee el menú administrativo.
Arriba

Botón Generar Reporte de Empleados

Este botón nos permite generar un reporte que se puede imprimir sobre todos los empleados de la institución o sobre un empleado en especifico.
Arriba
